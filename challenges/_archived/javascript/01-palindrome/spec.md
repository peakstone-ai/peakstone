# Palindrome check

Implement an ES module **`solution.js`** that exports a function:

```js
export function isPalindrome(s) { /* ... */ }
```

It returns `true` if `s` reads the same forwards and backwards, **ignoring case and any
non-alphanumeric characters**, and `false` otherwise.

Examples:
- `isPalindrome("A man, a plan, a canal: Panama")` → `true`
- `isPalindrome("race a car")` → `false`
- `isPalindrome("")` → `true`
