# Task

Implement a file **`solution.py`** with two functions:

- `initials(name)` — return the uppercase initials of each whitespace-separated word in `name`
  (e.g. `"ada lovelace"` → `"AL"`).
- `repeat_join(items, times)` — return the list `items` repeated `times` times, joined into a
  single string with `", "` (e.g. `(["a", "b"], 2)` → `"a, b, a, b"`).

Follow **all** conventions in AGENTS.md.
